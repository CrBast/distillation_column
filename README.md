# Arduino heating system project
Heating System Concept - Arduino Uno &amp; WPF

## The purpose
1. Manage the temperature of a resistor
2. Data transfer between the Arduino and the computer

## Arduino components
- **LCD screen**
- **Thermal Sensor** (*DS18B20 - Dallas*)
- **Transistor** (*BD675*)
- **100 Î© resistance**
- **2 x LED**
